== Description ==

This plugin allows for one-click install/activation of each plugin run by Qody's owls.

Arguably better than the default Wordpress plugin manager, the ease of use is fantastic : ) 

For more information about Qody's Owl Emporium or it's various prestigious workers, head on over to http://plugins.qody.co for the latest scoop.

== Installation ==

1. Upload `qodys-owl-emporium` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit the "Qodys Emporium" menu in the left sidebar of the admin area
